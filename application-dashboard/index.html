Cloud Sync (shared team)

When on, the entire dashboard (applications + sessions) is synced to a JSON file in the script owner's Google Drive via your Apps Script. Anyone you share the dashboard URL with — and who pastes the same Web App URL + secret — sees the same data.

Caveats: requires re-deploying Apps Script with the new Drive permission. Last write wins (no conflict resolution). Your Adzuna keys, Gmail OAuth client, and per-device prefs are NOT synced.
